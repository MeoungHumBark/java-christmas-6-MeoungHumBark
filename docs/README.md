1. 특별 할인
   - 별이 있는 날짜
2. 크리스마스 디데이 할인
3. 주중 주말 할인
   - 주중 할인
   - 주말 할인
4. 증정
----------------------------------------
### 총 구매 금액
1. 총 금액
2. 주문한 디저트 개수
3. 주문한 메인 메뉴 개수

1. 예상 방문 날짜
2. 메뉴와 개수
